Welcome to the Projects Repository

This repository contains various projects, including the A/B Testing Project, which is focused on analyzing e-commerce strategies through statistical methods.

About A/B Testing Project
The A/B Testing Project explores how different strategies in advertising can impact key metrics like conversion rates and user engagement. 
The goal is to apply statistical techniques, such as the Shapiro-Wilk test, Levene's test, and T-tests, to determine which variations of a strategy perform better.

So far, the A/B Testing analysis involves:

*Comparing multiple ad variations

*Performing data validation and statistical tests

*Drawing actionable insights for optimization

The problem statement and relevant details can be found in the respective files. The datasets used for analysis are also provided.
